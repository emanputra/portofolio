Penjelasan singkat mengenai WWW, mekanisme kerja, dan contoh struktur dasar dokumen HTML.

1. Apa itu WWW (World Wide Web)?

WWW (World Wide Web) adalah kumpulan situs di seluruh dunia yang memberikan layanan kepada pengguna untuk mengakses berbagai informasi melalui halaman web.

2. Mekanisme Kerja World Wide Web

Secara garis besar:

  1. Client (mis. browser) membuka URL atau alamat halaman.
  2. Browser mengirimkan permintaan (HTTP Request) ke server web.
  3. Server memproses permintaan dan mengirimkan balasan (HTTP Response) berisi HTML/CSS/JS atau resource lain.
  4. Browser menerima respon dan merender (menampilkan) halaman kepada pengguna.

Contoh Alur Sederhana

1. Client membuka browser.
2. Browser mengirimkan permintaan ke server melalui protokol HTTP.
3. Server memproses permintaan dan mengirimkan halaman HTML kembali ke browser.
4. Browser menampilkan halaman tersebut kepada pengguna.

3. Struktur Dasar HTML

Contoh struktur HTML paling sederhana:

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    Konten Halaman Web
  </body>
</html>

4. Contoh Lengkap (Dengan Meta dan Bahasa)

Gunakan template ini sebagai titik awal untuk membuat halaman web:

<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Contoh Halaman Web</title>
  <!-- Tambahkan link CSS atau script JS di sini -->
</head>
<body>
  <header>
    <h1>Selamat datang di situs saya</h1>
  </header>
  <main>
    <p>Ini adalah contoh paragraf pada halaman web.</p>
  </main>
  <footer>
    <small>&copy; 2025 â€” Contoh Situs</small>
  </footer>
</body>
</html>

5. Tips Singkat

  • Selalu sertakan <meta charset="utf-8"> untuk menghindari masalah encoding.
  • Gunakan lang di tag <html> untuk aksesibilitas dan SEO.
  • Gunakan <meta name="viewport"> agar halaman responsive pada perangkat mobile.
